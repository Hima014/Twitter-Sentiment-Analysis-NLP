# Twitter Sentiment Analysis using NLP and Machine Learning

## Project Overview

This project analyzes Twitter data to identify public sentiment toward different brands and topics using Natural Language Processing (NLP) and Machine Learning.

The project includes:

- Data Cleaning
- Exploratory Data Analysis
- Text Preprocessing
- Feature Engineering using TF-IDF
- Logistic Regression Classification
- Model Evaluation
- Business Insights

---

## Dataset

Twitter Entity Sentiment Analysis Dataset

Columns:

- Tweet ID
- Entity
- Sentiment
- Tweet

## Results

The Logistic Regression model achieved strong sentiment classification performance using TF-IDF features.

Evaluation metrics include:

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
